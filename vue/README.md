# game-of-life

A Vue version of Conway's "Game of Life" with Typescript, Vite and Tailwind CSS.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn dev
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
